# a1-progress — Learning Log

a1-progress is read-only (status + routing only, no plan/execute/modify actions),
so it has no mandatory Retro gate like a1-analyze or a1-fix. This file stays
empty by default. If a future run surfaces something worth carrying forward
(e.g. a routing recommendation that turned out wrong), append an entry here
in the same format as the other skills' `_learning.md` files.
